## История

Язык **[Java](https://ru.wikipedia.org/wiki/Java)** был разработан компанией **Sun Microsystems** под руководством **[Джеймса Гослинга](https://ru.wikipedia.org/wiki/%D0%93%D0%BE%D1%81%D0%BB%D0%B8%D0%BD%D0%B3,_%D0%94%D0%B6%D0%B5%D0%B9%D0%BC%D1%81)**.
* **1991 год**: началась работа над проектом **Green**, в рамках которого был создан язык программирования, получивший первоначальное название **Oak**.
* **1995 год:** название **Oak** было изменено на **Java**, так как оно уже было занято патентом.
* **23 мая 1995 года:** представлен первый релиз языка — **Java 1.0 (J2SE)**.
* **1998 год:** вышла версия **Java 2**, которая включала расширения для корпоративных приложений (**J2EE**) и мобильных устройств (**J2ME**).
* **2004 год:** была выпущена версия **Java 5 (J2SE 5.0)**, которая ввела **generics**, **аннотации** и улучшенные возможности для **многозадачности**.
* **2009 год:** компания **Oracle Corporation** приобрела **Sun Microsystems**, включая все технологии, связанные с языком Java.
* **2014 год:** вышла версия **Java 8**, которая привнесла лямбда-выражения, Stream API, новую библиотеку для работы с датами и временем (java.time) и методы по умолчанию в интерфейсах, значительно улучшив функциональность и производительность языка.
* **2017 год:** Java начала выпускать обновления каждые полгода, что ускорило развитие языка и его экосистемы.
